# \## 📖 Sobre o Projeto

# 

# YTGrabber é uma ferramenta completa de linha de comando desenvolvida em Python para buscar, reproduzir e baixar conteúdo do YouTube e YouTube Music. Com interface intuitiva 100% em português, suporte a MPV player integrado, conversão automática para MP3, e gerenciamento organizado de downloads por tipo (vídeos, playlists, álbuns e canais).

# 

# Ideal para quem quer ter controle total sobre seus downloads de áudio do YouTube e YouTube Music, sem precisar de interfaces gráficas complicadas ou serviços online duvidosos.

# 

# ---

# 

# \## ✨ Recursos

# 

# \- 🔍 \*\*Busca integrada\*\* no YouTube e YouTube Music com resultados organizados

# \- ▶️ \*\*Reprodução direta\*\* de vídeos e músicas usando MPV player embutido

# \- 💾 \*\*Download de áudio em MP3\*\* com conversão automática via FFmpeg

# \- 📦 \*\*Suporte a downloads em massa\*\* de playlists, álbuns e canais completos

# \- 🍪 \*\*Sistema de cookies\*\* para acesso a conteúdo restrito do YouTube Music

# \- 🇧🇷 \*\*Interface 100% em português\*\* com navegação intuitiva por menus

# \- 📁 \*\*Organização automática\*\* de downloads em pastas separadas (downloads, playlists, albuns, canais)

# \- ⚡ \*\*Leve e rápido\*\* - Interface CLI sem consumo excessivo de recursos

# \- 🔒 \*\*Privacidade\*\* - Tudo roda localmente, sem envio de dados para terceiros
